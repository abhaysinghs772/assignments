
## Installation

```bash
$ git clone https://github.com/abhaysinghs772/assignments.git
```

```bash
$ cd assignments/
```

```bash
$ git checkout -b tickete origin/ticket && cd tickete/ 
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
